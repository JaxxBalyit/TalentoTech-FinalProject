## Uso en código 🧩

### Instalación ⌨️

Para instalar las dependencias del proyecto, ejecutar el siguiente comando:

```bash
npm install
```

### Ejecución 💻

Para ejecutar el proyecto en modo desarrollo, ejecutar el siguiente comando:

```bash
npm run dev
```
